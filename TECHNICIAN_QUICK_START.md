# 🚀 QUICK START: Membuat Akun Teknisi

## Langkah Singkat

### **1️⃣ Login Admin**
```
Buka: http://localhost/login
Email: admin@example.com
Password: (password admin)
```

### **2️⃣ Akses Form Tambah Teknisi**
```
Menu: Kelola Teknisi → Tambah Teknisi
Atau: http://localhost/technicians/create
```

### **3️⃣ Isi Form (Contoh)**
```
Nama:          Budi Santoso
Telepon:       08123456789
Email:         budi.santoso@technic.id
Alamat:        Jl. Ahmad Yani No. 42, Jakarta
Spesialisasi:  AC Split, Maintenance
Password:      BudiPass123!
Status:        Tersedia
```

### **4️⃣ Klik Tombol "Simpan"** ✓

### **5️⃣ Lihat Notifikasi Sukses**
```
✓ Teknisi berhasil ditambahkan! User account telah dibuat.
Teknisi dapat login di /login/technician dengan email: budi.santoso@technic.id
```

## 🔓 Teknisi Login

### **URL Login:**

### **Credentials Contoh:**


## ⚠️ Penting!
- Email harus **unik** (belum terdaftar)
- Password minimal **6 karakter**
- Teknisi login di `/login/technician` (bukan `/login`)
